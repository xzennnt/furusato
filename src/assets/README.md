# Furusato Assets

Simpan gambar website di folder ini agar mudah diganti dari kode React.

- `furusato-training-map.png`: ilustrasi map sementara untuk bagian lokasi.
- Gambar hero bisa ditambahkan di folder ini, lalu import di `src/App.js`.
- Gambar galeri bisa ditambahkan di folder ini, lalu ganti placeholder `UPLOAD GAMBAR`.
